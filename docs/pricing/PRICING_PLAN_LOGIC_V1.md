# ZENTRA Pricing / Plan Logic V1

## Positioning
ZENTRA is not priced as a simple dashboard.
It is priced as an Economic Decision OS.

## Market Reference Bands
- Low BI tools: ~$10–20/user/month
- Advanced analytics tools: ~$70–150/user/month
- Trading/market tools: ~$60–200/user/month
- Professional terminals: $2,000+/month
- Enterprise risk/AI systems: custom annual contracts

## ZENTRA Pricing Principle
Price = product value + intelligence depth + report/audit weight + data cost + institutional usage.

## Product Plans

### Free
Price: $0
Purpose: acquisition
Includes:
- Free Feed
- Markets public view
- limited assistant guidance
- no full report

### Core
Suggested Price: $49–99/month
Target:
- individual professional
- small business user
Includes:
- 1 cockpit
- 1 product line
- basic report
- assistant guidance

### Expert
Suggested Price: $199–499/month
Target:
- professional user
- analyst
- active finance/risk user
Includes:
- Risk / FT / Credit access
- scenario engine
- conflict resolver
- simulation engine
- exposure engine
- reports

### Institutional
Suggested Price: $1,500–7,500/month
Target:
- SME / institution / fund / advisory desk
Includes:
- portfolio cockpit
- audit trace
- custom reports
- assistant workflows
- multi-user use
- compliance/disclosure layer

### Enterprise / Custom
Suggested Price: $25,000+/year
Target:
- bank
- insurer
- logistics / trade institution
- public-sector style institutional workflow
Includes:
- custom modules
- API/data integration
- governance
- role/access logic
- dedicated report packs

## Package Mapping
- Risk Intelligence Pack: Core / Expert / Institutional
- Financial Trade Pack: Expert / Institutional
- Credit Intelligence Pack: Core / Expert / Institutional
- Portfolio Intelligence Pack: Institutional / Enterprise

## Update Rule
Pricing is reviewed monthly against:
- market software prices
- data/API cost
- model/API cost
- report complexity
- customer segment
- institutional demand
