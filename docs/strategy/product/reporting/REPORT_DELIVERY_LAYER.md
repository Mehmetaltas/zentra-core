# ZENTRA — REPORT DELIVERY LAYER

## Durum
ACTIVE — STARTED

---

## Amaç

ZENTRA raporlarını sadece ekranda göstermek değil;
kullanıcının ihtiyacına göre farklı formatlarda, farklı seviyelerde ve farklı erişim mantıklarıyla üretmek.

---

# 1. Interaction Layer

Durum: ACTIVE

Rapor içinde kullanıcı:

- simple / halk dili
- technical / teknik dil
- executive / yönetici dili

modları arasında geçebilir.

---

# 2. User Request Based Reporting

Durum: ACTIVE

Kullanıcının girdiği bilgi isteğine göre rapor şekillenir:

- risk raporu
- kredi raporu
- trade raporu
- portfolio raporu
- executive summary
- detailed expert report
