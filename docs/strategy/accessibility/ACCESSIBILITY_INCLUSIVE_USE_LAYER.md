# ZENTRA — ACCESSIBILITY / INCLUSIVE USE LAYER

## Durum
ACTIVE — STARTED

---

## Amaç

ZENTRA'nın engelli, yaşlı, teknik bilgisi düşük veya farklı kullanım ihtiyacı olan kullanıcılar tarafından erişilebilir olmasını sağlamak.

Bu katman:

- ZENTRA Intel
- ZENTRA General AI
- Cockpit
- Assistant
- Report
- Token / Access
- Legal / Governance

genelinde uygulanır.

---

# 1. Visual Accessibility

Durum: ACTIVE

Görme zorluğu olan kullanıcılar için:

- yüksek kontrast modu
- büyük yazı modu
- screen reader uyumu
- klavye ile kullanım
- görsel öğelere metin karşılığı
- grafik / indicator için metin açıklaması

---

# 2. Audio / Hearing Accessibility

Durum: ACTIVE

İşitme zorluğu olan kullanıcılar için:

- sesli çıktı varsa yazılı karşılığı
- video / ses içeriklerinde altyazı
- kritik uyarıların sadece sesle verilmemesi
- görsel uyarı / metin uyarı

---

# 3. Cognitive Accessibility

Durum: ACTIVE

Teknik bilgisi düşük veya karmaşık sistemleri kullanmakta zorlanan kullanıcılar için:

- halk dili modu
- adım adım yönlendirme
- kısa açıklama
- teknik açıklamaya geçiş
- "ne yapmalıyım?" cevabı
- assistant guided usage

---

# 4. Motor Accessibility

Durum: ACTIVE

Fiziksel kullanım zorluğu olan kullanıcılar için:

- klavye navigasyonu
- büyük tıklama alanları
- minimum hassas hareket gereksinimi
- form sadeleştirme
- hata sonrası kolay geri dönüş

---

# 5. Language / Level Adaptation

Durum: ACTIVE

Assistant ve report çıktıları:

- simple / halk dili
- technical / teknik dil
- executive / yönetici dili
- TR / EN / AR

modlarında çalışır.

---

# 6. Screen Reader Standard

Durum: ACTIVE

Cockpit ve report:

- başlık hiyerarşisi
- aria-label mantığı
- text alternative
- table açıklamaları
- chart açıklamaları

taşır.

---

# 7. Accessibility Governance

Durum: ACTIVE

Accessibility yalnızca UI konusu değildir.

ZENTRA:

- erişilebilirlik hatasını issue olarak görür
- self-healing / audit sistemine yazar
- kullanıcı etkisini ölçer
- gerekiyorsa compensation / service credit politikasına bağlar

---

# 8. Legal / Compliance Connection

Durum: ACTIVE

Bu katman:

- WCAG uyumluluğu
- European Accessibility Act yaklaşımı
- ADA dijital erişilebilirlik yaklaşımı
- yerel erişilebilirlik ve tüketici hakları

ile uyumlu planlanır.

---

## Kilit Kural

ZENTRA erişilebilirlik sunan ek özellik değil,
erişilebilir doğan sistem olmalıdır.

---

## Kullanıcı Deneyimi Cümlesi

Kullanıcı sistemi bilmese de,
görmekte veya anlamakta zorlansa da,
assistant onu sade dille yönlendirir ve sistem kullanılabilir kalır.
